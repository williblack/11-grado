Objetivo de Aprendizaje

Aplicar principios de Arquitectura de la Información y UX/UI (User Experience / User Interface) para transformar datos crudos en interfaces de consulta jerarquizadas, funcionales y estéticamente optimizadas para la toma de decisiones.

📝 COPIAR AL CUADERNO

Fundamentos de UI en Bases de Datos

Conexión: Tras aprender sobre Diseño y Formatos de Consulta, ahora estructuraremos su presentación visual.

  • 1.

    Jerarquía Visual (Visual Hierarchy): Es el uso de tamaño, peso tipográfico (negrita) y posición para guiar el ojo del usuario hacia los datos más críticos (Ej: Totales, KPIs, Alertas).

  • 2.

    Teoría del Color Funcional: Uso de fondos neutros (Low cognitive load) para los datos, reservando colores saturados solo para encabezados y estados (Rojo = Error, Verde = Aprobado).

  • 3.

    Micro-componentes (Autoformas y Cuadros): Elementos flotantes que actúan como "Llamados a la Acción" (CTAs). Transforman una hoja de cálculo estática en una aplicación interactiva (Menús, Botones de Macro).

🖥️ TRABAJO EN LA SALA

Laboratorio Práctico: IA + Arquitectura de Datos en Excel

Paso 1: Generación de Datos con Inteligencia Artificial

Copia el siguiente Prompt y pégalo en una IA (ChatGPT, Gemini, Claude) para generar tu Base de Datos cruda.

"Actúa como un analista de datos. Genera una tabla con 12 filas y 5 columnas sobre el 'Inventario de Equipos Tecnológicos del Colegio'. Las columnas deben ser: Código_Equipo, Tipo (Laptop/PC/Tablet), Marca, Año_Compra, Estado (Operativo/Mantenimiento/Baja). Entrega ÚNICAMENTE la tabla en formato markdown puro para que yo pueda copiarla y pegarla directamente en Excel."

Paso 2: Renderizado Visual en Excel

Pega los datos en Excel y aplica el siguiente flujo de trabajo profesional (UX/UI):

  1. Ajuste Estructural: Ajusta el ancho de columnas automáticamente. Alinea textos al medio.
  2. Jerarquía: Cambia la tipografía de los encabezados (Ej: 'Segoe UI' o 'Montserrat'), tamaño 12, Negrita. Letra color blanco.
  3. Color Funcional: Fondo de encabezados Azul Oscuro o Gris Pizarra. Fondo de filas intercaladas (Zebra striping) en gris muy claro.
  4. UI Components (Autoformas): Ve a Insertar > Formas. Dibuja un rectángulo con esquinas redondeadas en la parte superior. Escribe "Panel de Búsqueda" y dale un efecto de sombra sutil.
